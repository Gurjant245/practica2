Navegador web

Un navegador web (en ingles, web browser) es un software, aplicaciones o programa que permite el acceso a la Web, interpretando la informacion de distintos tipos de archivos y sitios web para que estos puedan ser vistos.

La funcionalidad basica de un navegador web es permitir la visualizacion de documentos de texto, posiblemente con recursos multimedia incrustados.

Ejemplos de navegador web

Safari: Es un navegador web de codigo cerrado desarollado por Apple Inc. Esta disponible en macOS, iPadOS e iOS.

Mozilla firefox es un navegador web libre de codigo abierto desarollado para Linux, android, IOS, macOS y Microsoft Windows coordinado Por la Corporacion Mozilla y la Fundacion Mozilla.

Google Chrome: es un navegador de codigo cerrado desarollado por Google, aunque derivado de proyectos de codigo abierto.
